# socialme

GRADUATION PROJECT
SOCIAL ME APP
Project Name: SocialMe - Free Multi-Platform Social Media Application

This project aims to provide free communication and sharing opportunities
among students through the eponymous "SocialMe" application. The
application stands out with its multi-platform support that can be used both on
mobile devices and on the web. Unlike the censorship and restrictions that are
common in today's social media platforms, "SocialMe" aims to provide
students with a completely free environment.
